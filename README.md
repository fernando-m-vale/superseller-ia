# Super Seller IA

Plataforma de IA para otimizar anúncios e operações em marketplaces como Shopee, Mercado Livre, Amazon e Magalu.

## MVP Goals
- Conectores Shopee e Mercado Livre
- Health Score por anúncio
- Action Engine (recomendações + execução assistida)
- Relatórios automáticos

## Tech Stack
- Next.js
- Node/Python
- AWS (Lambda, S3, RDS)
- Terraform
- GitHub Actions
